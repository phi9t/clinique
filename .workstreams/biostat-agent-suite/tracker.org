* Overview
Implements RFCs 0000-0006 (see docs/rfcs/). Order is by *validation feasibility*
(see design.md): substrate -> RFC-0003 -> RFC-0002/0005 -> RFC-0004 -> RFC-0001.
Stack: Python 3.14 via uv; R engines via pinned Docker image; tests via pytest.

** Status (2026-05-23)
All RFCs implemented at scaffold+ level and green: =uv run pytest= -> 47 passed, 3 skipped
(rpact Docker cross-checks skip until the daemon is up and the image is built). =ruff= clean.
Done: substrate (0000), RFC-0003 (full), RFC-0001 (rules+seeded-defect), RFC-0002 side-lock,
RFC-0005 data-wall, RFC-0004 conformance triage. Deferred: rpact cross-check (docker daemon),
public-data fixture wiring (pharmaverse / CDISC Pilot / CORE), natural-defect benchmark (partner).

Invariants enforced as hard pass/fail (not metrics):
- numeric-provenance (RFC-0000 s7)
- validated-engine-only computation (RFC-0003)
- reproducibility / determinism
- (later) side-lock (0002), data-wall (0005), read-only (0001)
